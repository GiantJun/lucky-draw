## 年会抽奖

[魔改版演示地址](https://GiantJun.github.io/lucky-draw/
)

## 功能特色

- 支持基于人名的抽奖，区分员工类型（正职/外包）
- 支持 CSV 文件批量导入人员名单和奖品配置
- 支持额外抽奖次数功能，提升中奖概率
- 灵活的中奖规则配置，可控制同一人多次中奖行为
- 多等级奖品配置（特等奖/一等奖/二等奖/三等奖）
- 每个奖品可独立配置参与人员类别

## 使用说明

### 第一步：准备CSV文件

#### 1. 人员名单 CSV 格式

文件需包含 3 列：人名、员工类型、额外抽奖次数

```csv
人名,员工类型,额外抽奖次数
张三,正职,2
李四,外包,0
王五,正职,1
```

字段说明：
- **人名**：参与抽奖人员的姓名（必填）
- **员工类型**：`正职` 或 `外包`（必填）
- **额外抽奖次数**：该人员可额外参与抽奖的次数（必填，0 表示只参与 1 次）
  - 例如：张三配置为 2，则候选名单中会出现 "张三1"、"张三2" 两个名字

#### 2. 奖品配置 CSV 格式

文件需包含 4 列：奖品等级、奖品名称、数量、参与人员类别

```csv
奖品等级,奖品名称,数量,参与人员类别
特等奖,iPhone 15 Pro,1,正职
一等奖,iPad Air,3,正职
二等奖,小米手环,5,外包
三等奖,保温杯,10,全部
```

字段说明：
- **奖品等级**：`特等奖`、`一等奖`、`二等奖`、`三等奖`（必填）
- **奖品名称**：具体的奖品名称（必填）
- **数量**：该奖品的数量（必填，数字）
- **参与人员类别**：`正职`、`外包` 或 `全部`（必填）

**提示**：项目根目录下提供了 `人员名单示例.csv` 和 `奖品配置示例.csv` 供参考。

### 第二步：配置抽奖

#### 方式一：快速开始（推荐新手）

1. 点击右上角 **"抽奖配置"** 按钮
2. 直接点击 **"加载示例数据"** 按钮，系统会自动导入项目自带的示例人员和奖品配置
3. 查看导入结果，确认无误后点击 **"保存配置"**
4. 即可开始体验抽奖功能

#### 方式二：自定义配置

1. 点击右上角 **"抽奖配置"** 按钮
2. 设置抽奖标题（可选）
3. 配置功能开关：
   - **额外抽奖次数**：开启后使用 CSV 中配置的额外抽奖次数（默认开启）
   - **一人中奖全部失效**：开启后同一人的所有候选名只能中一个奖（默认开启）
4. 点击 **"导入 CSV"** 按钮，分别导入人员名单和奖品配置
   - 如果已有数据，导入前会提示是否覆盖
5. 查看导入结果，确认无误后点击 **"保存配置"**

### 第三步：开始抽奖

1. 点击右侧 **"开始"** 按钮
2. 在弹出的对话框中：
   - 选择 **奖品等级**（特等奖/一等奖/二等奖/三等奖）
   - 选择 **具体奖品**
   - 查看奖品剩余数量和符合条件的候选人数
   - 选择 **抽取方式**（1人/5人/一次抽完/自定义）
3. 点击 **"立即抽奖"**，3D 标签云开始滚动
4. 点击 **"停止"**，显示中奖结果

### 功能开关详解

#### 1. 额外抽奖次数功能

- **开启**：使用 CSV 中配置的额外抽奖次数，候选名单中会出现 "张三1"、"张三2" 等
- **关闭**：忽略额外抽奖次数配置，所有人只显示真实姓名

#### 2. 一人中奖全部失效功能

- **开启（默认）**：张三1 中奖后，张三2 也不能再参与任何抽奖
- **关闭**：
  - 同一个奖品：张三1 中奖后，张三2 不能再抽这个奖品
  - 不同奖品：张三1 中奖后，张三2 还可以抽其他奖品

### 抽奖结果

- 点击右上角 **"抽奖结果"** 查看所有中奖记录
- 按奖品等级和奖品名称分组显示
- 点击中奖人员卡片可删除该中奖记录，删除后该人员可继续参与后续抽奖

### 重置功能

- 点击右侧 **"重置"** 按钮
- 可选的重置选项：
  1. 重置全部数据
  2. 重置抽奖配置
  3. 重置人员名单
  4. 重置奖品配置
  5. 重置照片
  6. 重置抽奖结果

### 导入照片（可选）

- 点击右侧 **"导入照片"** 按钮
- 按照人名-照片的一对一导入
- 照片格式支持 `.jpg` 和 `.png`，照片大小不能超过 `150kb`，建议 `20-50kb`，建议尺寸为 `160*160px`
- 抽奖结果将以照片形式展示

## 温馨提示

- 本抽奖程序无暗箱操作，无后台，无后门。
- 名单和照片显示只需导入一种即可,无导入数据则使用抽奖号码。
- 建议使用最新的 Chrome 浏览器打开体验最佳。
- 由于背景音乐加载较慢，可以在抽奖前提前打开缓存好。或者安装 windows 版本软件到本地，无需联网即可使用。
